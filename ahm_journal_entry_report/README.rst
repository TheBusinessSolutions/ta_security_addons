========================================
Journal Entry Report for Odoo 17
========================================
**Developed by Ahson Mahmood**

A clean, professional PDF report for Journal Entries in Odoo Accounting.

This module lets accountants and finance users generate neatly formatted Journal Entry PDFs
with dynamic company color styling, analytic distribution details, and properly aligned debit/credit totals.

---

Features
========
* Print professional PDF Journal Entry reports directly from Odoo.
* Includes all major fields: Account, Label, Partner, Analytic Distribution, Debit, Credit, Amount Currency.
* Displays company, journal, and reference details in the header.
* Computes Debit and Credit totals automatically.
* Adopts your company's color scheme dynamically in the report.
* Easy one-click printing from the Journal Entries list or form view.
* Simple, lightweight, and fully compliant with Odoo 17 Accounting.

Installation
============
1. Copy the module folder `ahm_journal_entry_report` into your Odoo `addons` directory.
2. Restart your Odoo service.
3. Update the apps list.
4. Install **Journal Entry Report** from the Apps menu.
