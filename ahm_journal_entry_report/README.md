# Journal Entry Report (PDF)

Author: **Ahson Mahmood**  
Website: https://www.linkedin.com/in/ahsonmahmood/  
License: **LGPL-3**

This module adds a printable PDF report for Journal Entries (`account.move`).

## Features
- Print from the Journal Entries list (list action / Report)
- Header: Company, Journal, Date, Number, Reference, Partner
- Lines table: Account, Label, Debit, Credit, Amount Currency, Balance
- Totals for Debit, Credit and Balance

## Compatibility
Tested with Odoo 15/16/17 Community.

## Installation
1. Copy `ahm_journal_entry_report` into your Odoo addons path.
2. Update Apps List and install.
3. Open **Accounting > Accounting > Journal Entries**, select a record, and print **Journal Entry (PDF)**.
