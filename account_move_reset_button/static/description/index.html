Account Move Reset Button

Description

This module is configured to create a user group called "Can reset account move". Only the members of this group will be able to view the "Reset to Draft button".

Configuration

Go to Settings - - > Users & Companies - - > Users. Select Admin user.

“Can reset account move” boolean group is created under the Access Rights tab of the user for this functionality.


Functionality

Once the “Can reset account move” group is enabled for the user, Only the members of this group will be able to view the "Reset to Draft button" on the invoices, credit note, bills, debit note and journal entries.

Once the “Can reset account move” group is disabled for the user then the “Reset to Draft” button on the invoices, credit notes, bills, debit notes, and journal entries is not visible for them. So that the user doesn't move the invoice to Draft state once it is posted.
