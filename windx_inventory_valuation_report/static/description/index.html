General

What does this module do?

It generates audit-ready Inventory Valuation reports in PDF/XLS with multi-warehouse/location support, category-wise grouping, and product/category filters.

Does it work with multiple warehouses and locations?

Yes. You can roll up company-wide values or drill down by warehouse, internal location, or bin.

Can I filter by specific products or categories?

Yes. Filter by product, category, or both; combine with warehouse/location filters as needed.

Is it compatible with multi-company setups?

Yes. Run per company according to user access rights and selected company context.
