Restrict operation/move on location

This module empowers warehouse managers to restrict stock moves involving specific locations. For example, a manager might want to prevent standard users from performing moves or operations involving 'Inventory Loss' or 'Scrapped' locations. This module allows you to designate authorized users who are permitted to manage inventory and move stock to these restricted locations, ensuring better control and security.
