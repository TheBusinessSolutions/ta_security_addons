Update Qty Button Access

Overview

  • When the "On Hand Qty Update Access" setting is active, users with the appropriate permissions will see the "Update Quantity" button on the product screen. This button enables users to adjust the inventory levels for the product, facilitating real-time inventory management. If the access setting is not enabled, the button will be hidden, restricting users from making changes to the on-hand quantity. This functionality ensures that only authorized personnel can modify inventory levels, helping maintain accurate stock records and control over inventory adjustments.

    • Before Access Setting is Enabled

    Button Visibility

  • The "Update On Hand Quantity" button is not visible on the product screen.

    • After Access Setting is Enabled

    Button Visibility

  • The "Update On Hand Quantity" button appears on the product screen for users with the appropriate permissions.
